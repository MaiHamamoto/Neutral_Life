# 【概要】
- Neutral_Life(サイト名)
- ECサイトのポートフォリオです。

# 【構成言語】
- PHP
- twig(HTML)
- scss(CSS)
- JavaScript(jQuery, Vue.js)
- MySQL
- (※Vue.jsの価格昇降順機能も実装したが、何らかの不具合により動作しておらず、現在原因解析中)
- ※MAMPのhtdocs配下で動作確認済み

# 【実装機能説明】
- 会員登録機能
- ログイン機能
- カート追加機能
- ログイン後名前を表示させる機能(PHP)を実装
- UI/UXデザイン 
- jsによるスクロールアニメーション
- レスポンシブ対応を実装
- 商品キーワード検索(色別、単語別)
- 商品色別分類(白、黒、灰色別)

# 【コンセプト】
- モノトーンのプロダクトのセレクトショップのECサイトを想定して作成
- 対象ターゲット層：20〜40代のモード好きでおしゃれでシンプルなものが好きな層
